⚡ Démarrage Rapide

Déployez et exécutez ces agents en local en moins de deux minutes.

1

Cloner le projet

git clone https://github.com/theriderymarketing/antigravity-agent-project.git
2

Setup venv & packages

cd antigravity-agent-project && python3 -m venv venv && source venv/bin/activate && pip install -r requirements.txt
3

Lancer le serveur API local

python server.py
